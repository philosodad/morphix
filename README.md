Tiny library of functions to perform transforms on maps and arrays.

## Installation

The package can be installed from hex.pm

Add `:morphix` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:morphix, "~> 0.8.0"}
  ]
end
```
